# 枚举



## 特殊个性
### 1. 直接将数据附加到枚举的每个变体上
```rust
#[derive(Debug)]
enum IpAddrKind {
    /**
     * 直接将数据附加到枚举的每个变体上, 仅仅使用枚举并将数据直接放进每一个枚举变体
     * ---> IpAddr 枚举的新定义表明了 V4 和 V6 变体都关联了 String 值
     */
    V4(String),
    V6(String),
}

fn main() {
    let home = IpAddrKind::V4(String::from("127.0.0.1"));
    let home2 = IpAddrKind::V4(String::from("127.0.0.2"));
    let loopback = IpAddrKind::V6(String::from("::1"));

    println!("home detail: {:#?}", (home));
    println!("home2 detail: {:#?}", (home2));

    println!("loopback detail: {:#?}", (loopback));
}

```
