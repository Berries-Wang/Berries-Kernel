## 模块系统（the module system）
模块系统包含以下内容:

|名称|说明|备注|
|-|-|-|
|- 包（Packages）|- Cargo 的一个功能，它允许你构建、测试和分享 crate|-|
|-|-|-|
|- Crates|- 一个模块的树形结构，它形成了库或可执行文件项目|-|
|-|-|-|
|- 模块（Modules）和 use|- 允许你控制作用域和路径的私有性|-|
|-|-|-|
|- 路径（path）|- 一个为例如结构体、函数或模块等项命名的方式|-|
|-|-|-|
