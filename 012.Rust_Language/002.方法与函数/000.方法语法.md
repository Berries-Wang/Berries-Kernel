# 方法语法
- 在一个 impl 块中，Self 类型是 impl 块的类型的别名
- 方法的第一个参数必须有一个名为 self 的Self 类型的参数，所以 Rust 让你在第一个参数位置上只用 self 这个名字来简化
- 方法可以选择获得 self 的所有权，或者像我们这里一样不可变地借用 self，或者可变地借用 self，就跟其他参数一样。

## 示例代码
```rust
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    /**
     * 在 area 的签名中，使用 &self 来替代 rectangle: &Rectangle， &self 实际上是 self: &Self 的缩写。
     * 在一个 impl 块中，  Self 类型是 impl 块的类型的别名。 
     *     方法的第一个参数必须有一个名为 self 的Self 类型的参数， 所以 Rust 让你在第一个参数位置上只用 self 这个名字来简化。
     *         注意，我们仍然需要在 self 前面使用 & 来表示这个方法借用了 Self 实例， 就像我们在 rectangle: &Rectangle 中做的那样。
     *     方法可以选择获得 self 的所有权，或者像我们这里一样不可变地借用 self，或者可变地借用 self，就跟其他参数一样。
     */
    fn area(self: &Self) -> u32 {
        self.width * self.height
    }

    /**
     * 自动引用和解引用（automatic referencing and dereferencing）的功能
     * 
     *  方法的接收者! , 没有&，所以，所有权会移动
     */
    fn width(self) -> bool {
        return self.width > 0;
    }
}

fn main() {
    let rect1 = Rectangle {
        width: 30,
        height: 50,
    };

    println!(
        "The area of the rectangle is {} square pixels.",
        area(&rect1)
    );

    println!(
        "The area of the rectangle is {} square pixels. ==> impl",
        rect1.area()
    );

    if rect1.width() {
        println!("The rectangle has a nonzero width; it is {}", rect1.width); // value borrowed here after move (值在此处被借用，但在之前已经被移动（move）了)
    }

    println!("rect1 is {:#?}", rect1);

    let scale = 2;
    let rect2 = Rectangle {
        width: dbg!(30 * scale),
        height: 50,
    };

    dbg!(&rect2);
}

fn area(rect: &Rectangle) -> u32 {
    rect.height * rect.width
}
```

## 关联函数
所有在 impl 块中定义的函数被称为 关联函数（associated functions），因为它们与 impl 后面命名的类型相关。

可以定义不以 self 为第一参数的关联函数（因此不是方法），因为它们并不作用于一个结构体的实例

### 示例代码
```rust
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    fn square(size: u32) -> Self {
        Self {
            width: size,
            height: size,
        }
    }
}

fn main() {
    /**
     * :: 语法用于关联函数和“模块”创建的命名空间
     */
    let rect_square: Rectangle = Rectangle::square(20);
    println!("Hi {:#?}", rect_square)
}

```
