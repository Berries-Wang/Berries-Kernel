# Rust
- Rust 是 静态类型（statically typed）语言，也就是说在编译时就必须知道所有变量的类型。

- 无 GC 且无需手动内存管理、性能高、工程性强、语言级安全性以及能同时得到工程派和学院派认可的语言 <sup>from “Rust语言圣经(Rust Course)”</sup>
  + 没有 GC 也无需手动内存管理,性能比肩 C++/C 还能直接调用它们的代码、安全性极高
