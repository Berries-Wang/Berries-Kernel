# dbg! 宏

调用 dbg! 宏会打印到标准错误控制台流（stderr）

## 示例代码
```rust
// 打印出调试信息的功能
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}

fn main() {
    let rect1 = Rectangle {
        width: 30,
        height: 50,
    };

    println!(
        "The area of the rectangle is {} square pixels.",
        area(&rect1)
    );

    println!("rect1 is {:#?}", rect1);

    let scale = 2;
    let rect2 = Rectangle {
        width: dbg!(30 * scale),
        height: 50,
    };

    dbg!(&rect2);
}

fn area(rect: &Rectangle) -> u32 {
    rect.height * rect.width
}

```
