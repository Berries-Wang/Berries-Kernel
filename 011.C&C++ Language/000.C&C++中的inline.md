# 为什么需要inline(内联)函数？
> 学习: <br/> - [001.UNIX-DOCS/037.内核基础/001.程序是怎么运行的/001.函数的调用_返回与栈.md](../001.UNIX-DOCS/037.内核基础/001.程序是怎么运行的/001.函数的调用_返回与栈.md)

使用内联函数可以加速程序运行，因为它们消除了与函数调用关联的开销,函数调用有哪些开销呢？
- 将返回地址推送到堆栈
- 将参数推送到堆栈
- 跳转到函数体
- 然后在函数完成时执行返回指令
