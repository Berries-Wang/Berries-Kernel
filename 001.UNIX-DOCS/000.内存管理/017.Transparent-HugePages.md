# Transparent HugePages(透明大页)


---

## 存在的性能问题
+ Transparent HugePages can cause memory allocation delays during runtime. To avoid performance issues, Oracle recommends that you disable Transparent HugePages on all Oracle Database servers. Oracle recommends that you instead use standard HugePages for enhanced performance.(“透明巨页（Transparent HugePages）可能导致运行时的内存分配延迟。为了避免性能问题，Oracle 建议在所有 Oracle 数据库服务器上禁用透明巨页。Oracle 建议您改为使用标准巨页（Standard HugePages）以获得更高的性能。”) <sup>[Disabling Transparent HugePages](./998.REFS/database-installation-guide-linux.pdf)#'Disabling Transparent HugePages'</sup>
