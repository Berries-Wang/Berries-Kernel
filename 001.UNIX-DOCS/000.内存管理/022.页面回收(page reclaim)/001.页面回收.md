|什么是页面回收|- 在Linux操作系统中: <br/> ++ 当内存充足时，内核会尽量多地使用内存作为文件缓存（page cache），从而提高系统的性能 <br/>++ 当内存紧张时，文件缓存页面会被丢弃，或者把修改的文件缓存页面回写到存储设备<sup>磁盘,慢</sup>中，与块设备同步之后便可释放出物理内存<br/>- 小总结: 无论系统中有多少内存，都是不够用的,对应内核处理策略:<br/>***页交换(swapping)**: Linux操作系统会使用存储设备作为交换分区，内核将很少使用的内存换出到交换分区，以便释放出物理内存|-|
