# QEMU
## 什么是 QEMU？
QEMU is a generic and open source machine emulator and virtualizer.(QEMU 是一款通用的开源机器模拟器与虚拟化器。)
- 机器模拟器 (Machine Emulator)： 它可以模拟完整的硬件系统（如 CPU、内存、外设）。这意味着你可以在你的 x86 电脑上运行为 ARM、PowerPC 或 MIPS 架构编写的程序。它通过“动态二进制翻译”技术，将目标平台的指令实时转换为本地 CPU 能理解的指令。
- 虚拟化器 (Virtualizer)： 当与 KVM 配合使用时，QEMU 可以以接近原生的性能运行虚拟机。在这种模式下，QEMU 负责模拟 I/O 设备（如硬盘、网卡、显卡），而复杂的 CPU 计算任务则直接交给底层的 KVM 和物理硬件处理。

QEMU can be used in several different ways. The most common is for System Emulation, where it provides a virtual model of an entire machine (CPU, memory and emulated devices) to run a guest OS. In this mode the CPU may be fully emulated, or it may work with a hypervisor such as KVM, Xen or Hypervisor.Framework to allow the guest to run directly on the host CPU.(QEMU 可以通过多种不同的方式使用。最常见的是 系统模拟 (System Emulation)，在这种模式下，它提供整个机器的虚拟模型（包括 CPU、内存和模拟设备）来运行客户操作系统（Guest OS）。在此模式下，CPU 既可以被完全模拟，也可以配合 KVM、Xen 或 Hypervisor.Framework 等虚拟机管理程序（Hypervisor）工作，从而允许客户机直接在宿主机 CPU 上运行。)


The second supported way to use QEMU is User Mode Emulation, where QEMU can launch processes compiled for one CPU on another CPU. In this mode the CPU is always emulated.(QEMU 支持的第二种方式是用户模式模拟 (User Mode Emulation)。在这种模式下，QEMU 可以运行为一种 CPU 架构编译的进程，而实际的运行环境是另一种 CPU 架构。在此模式下，CPU 始终是通过模拟方式工作的。)
