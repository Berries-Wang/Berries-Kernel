# 内存指标项
|指标|全称|含义|特点|
|-|-|-|-|
|VSS,Virtual Set Size,     |虚拟耗用内存,包含所有申请的空间，甚至是还没用的，数值最大。|
|RSS,Resident Set Size,    |常驻内存,实际在物理内存里的空间，最常用来观察进程状态。(不包括被交换到磁盘上的内存（Swap），也不包括虽然申请了但尚未使用的虚拟内存)|
|PSS,Proportional Set Size,|比例分配内存,会把共享库的大小按比例平分给使用它的进程，更公平。|
|USS,Unique Set Size,      |独占内存,进程自己私有的内存，不包含任何共享库，最能反映杀死该进程能释放多少内存。|
