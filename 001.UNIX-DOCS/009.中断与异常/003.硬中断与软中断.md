# 硬中断与软中断
|中断类型|区别|备注|
|-|-|-|
|硬中断（Hard Interrupt）|- 硬中断是由硬件设备产生的中断，用来通知CPU有重要的事件需要立即处理。|-|
|-|-|-|
|软中断（Soft Interrupt）|- 软中断是预留给系统中对时间要求较严格和重要的下半部使用的，而且目前驱动中只有块设备和网络子系统使用了软中断：[Run Linux Kernel (2nd Edition) Volume 1: Infrastructure:2.6.1　软中断](../../001.UNIX-DOCS/../007.BOOKs/Run%20Linux%20Kernel%20(2nd%20Edition)%20Volume%201:%20Infrastructure.epub) <br/>|-|
