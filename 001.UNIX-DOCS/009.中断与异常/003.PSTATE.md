# PSTATE
是一个硬件电路，保存了所有每一条汇编指令的运行结果. 参考:[硬件环境与软件抽象：ARM指令集架构](../000.内存管理/998.REFS/000.中山大学-操作系统/2-0303-hardware-isa.pdf)

- "保存了所有每一条汇编指令的运行结果" 怎么理解?
  ```assembly
      .L3:
        mul w0,w0,w2
        subs w1,w1,#1
        bne .L3
    // 这段代码的每一条指令执行完后，PSTATE 都会保存这条指令的运行结果，比如 subs 指令执行完后，PSTATE 会保存更新后的标志位（Z、N、C、V）。在执行 bne 指令时，会根据 PSTATE 中的标志位来决定是否跳转。 --- 即 PSTATE 保存了 subs 指令的结果，bne 指令根据 PSTATE 的结果来决定是否跳转。
    // bne 不等于时跳转
  ```
